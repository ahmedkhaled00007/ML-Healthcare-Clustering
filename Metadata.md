# Healthcare Dataset Metadata

This is the metadata (data dictionary) for the Kaggle Healthcare Dataset (prasad22/healthcare-dataset).

| Column             | Description / Meaning                                                                 |
|-------------------|---------------------------------------------------------------------------------------|
| Name               | Patient’s name.                                                                       |
| Age                | Age of the patient in years.                                                          |
| Gender             | Gender of the patient (“Male” / “Female”).                                           |
| Blood Type         | Patient’s blood group, e.g., “A+”, “O–”, etc.                                        |
| Medical Condition  | Primary diagnosis / medical condition (like Diabetes, Hypertension, etc.)           |
| Date of Admission  | The date when the patient was admitted to the hospital.                               |
| Doctor             | The name of the attending physician.                                                 |
| Hospital           | Name of the hospital / medical facility.                                             |
| Insurance Provider | Name of the insurance company covering the patient.                                   |
| Billing Amount     | The billed cost (monetary) for the patient’s hospital stay / treatment.             |
| Room Number        | Which room the patient was admitted to.                                              |
| Admission Type     | Category of admission: “Emergency”, “Urgent”, or “Elective”.                         |
| Discharge Date     | The date on which the patient is discharged.                                         |
| Medication         | The medication(s) given during this admission (e.g., “Aspirin”, “Penicillin”).      |
| Test Results       | The result of a medical test (“Normal”, “Abnormal”, or “Inconclusive”).              |

**Other metadata:**

- Number of records: 10,000 synthetic entries.  
- Data type: Mostly categorical (e.g., gender, blood type, condition) + some numeric (age, billing).
